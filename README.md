## Uppuru

Simple SPA anonymous filehosting.

### Used technologies

#### Backend
* Slim framework
* SphinxSearch
* MySQL
* GetID3

#### Frontend
* Vue.js
* Bootstrap

### Requirements
* PHP >= 7.1
* SphinxSearch
* MySQL
* Composer
* npm
