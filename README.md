## Uppuru

Simple SPA anonymous filehosting.

### Used technologies

#### Backend
* Slim framework
* SphinxSearch
* MySQL
* GetID3

#### Frontend
* Vue.js
* Bootstrap
* Vue-Aplayer
* Vue-Dplayer
* Vue-Moment

### Installation
* Clone repository `git clone https://github.com/h1royuki/filehosting.git`
* Run `make init`
* Open `127.0.0.1:8081`
